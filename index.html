Operadores lógicos

Eles podem ser usados para pensar de forma lógica sobre booleanos

  • && (and) "Ê" Seu resultado é true se ambos os valores forem verdadeiros
  • || (or) "OU" Produz um valor true se qualquer um dos valores forem verdadeiros.
  • ! (not) Inverte o valor que é dado a ele

Laços de repetição

Loop, repetir a mesma coisa várias vezes

Operadores de atribuição

  • = atribuição de valor
  • + atribuição de adição/concatenação
  • += atribuição de adição
  • - atribui uma subtração
  • -= atribuição de subtração
  • *= atribuição de multiplicação
  • *
  • /
  • /= atribuição de divisão
  • ++ atribuição de incremento. Operador unário. Adiciona 1 ao seu operador
  • -- atribuição de decremento. Operador unário. Subtrai 1 do seu operador.
